Generar código de acceso

Esta página es solo para vos (quien armó el sitio). No la compartas ni la dejes linkeada en ningún lado visible. Genera un archivo que reemplaza al token de GitHub por una contraseña simple para tu amigo.
Subí este contenido como el archivo data/access.json en tu repo (reemplazando el que exista). Guardá la contraseña en un lugar seguro — si la perdés, tenés que generar este código de nuevo con el mismo token.
